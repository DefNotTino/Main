Vending Machines (2021)

Vending Machines (2021)

Created using Aseprite and Picocad

This is a personal project of mine. I created it over the summer to explore the softwares Aseprite and Picocad to create animated 8-bit GIFs. I created this over the Summer of 2021 as a personal project. The color theme and design was tailored for San Jose State University (SJSU) as I wanted to imagine if I were designing a vending machine that people would use on campus. The first vending machine is a 3D render using Picocad. To create the color scheme and layout, I designed it first in Aseprite. Then I exported it to a Picocad file so it could register it. In Picocad, I designed a 3D box with each plane being covered by the design I created in Aseprite. The second image, is a 2D version I created using only Aseprite. The blinking effect was created using two frames toggled on and off.
