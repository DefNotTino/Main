The Dawn is Your Enemy

The Dawn is Your Enemy (2021)

The original screen size for this project is 800x800. Please click the image above to enlarge, or use the video link below to see the animation.
Created using Processing 3.4 and 4.1

This project was created during my time at Art 101 (SJSU). It was created using a computer program named, Processing. As our class learned more about Processing, we started diving into the advanced techniques, which are listed below. Interestingly enough, I found this to be the simplest and most enjoyable lecture so far. This is because I accidently created an amazing work of art using a combination of the radians function with the rotate function. In this work, there is a star orbiting another massive star. The idea came from the background code which I made each eclipse smaller using the mod function learned in exercise two. This is to replicate a bunch of smaller stars radiating in space.

  • Custom functions w/o arguments
  • Custom functions w/ arguments
  • Parent fucntions
  • Translation commands
  • pushMatrix(), popMatrix(), translate, rotate
